# 💼 Personal Portfolio & Calculator Website

This repository contains two web development projects created using **HTML, CSS, and JavaScript** as part of my learning journey in front-end web development.

## 📖 About the Project

The main project is a **Personal Portfolio Website** designed to showcase my profile, technical skills, educational background, hobbies, and contact information. It includes a modern dark theme with RGB glow effects, an animated welcome screen, and smooth transitions to create an engaging user experience.

The repository also includes a **Simple Calculator** that performs basic arithmetic operations with support for both button clicks and keyboard input.

---

## ✨ Portfolio Features

- Animated intro/welcome screen
- Modern dark UI with RGB glow effects
- About Me section
- Skills and programming languages
- Educational journey
- Project showcase
- Contact section
- Links to LinkedIn and Instagram
- Resume (CV) download option
- Responsive and clean layout
- Smooth animations and hover effects

---

## 🧮 Calculator Features

- Basic arithmetic operations
- Addition, subtraction, multiplication, and division
- AC (Clear) button
- Delete last character
- Keyboard input support
- Error handling for invalid expressions
- Simple and user-friendly interface
